### Dependencies

GNU Radio v3.7.

packets from your packet manager:
```
sudo apt-get install cmake libboost-all-dev liblog4cpp5-dev swig
```


### Installation

```
mkdir build
cd build
cmake ..
make
sudo make install
sudo ldconfig
```


### Usage

open apps/rds_rx.grc example flow graph in GNU Radio Companion.
